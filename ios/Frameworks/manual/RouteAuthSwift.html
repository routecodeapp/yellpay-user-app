Authentication (Swift)

Server Access

Note: In V1, a fixed site was provided, but now it supports access to multiple servers by specifying the server name.

RouteAuth Class

Description

Content

Package

com.platfield.unidsdk.routecode

Class Name

RouteAuthSwift

Overview

This class handles the registration of authentication keys and Unid login authentication.

info.plist

If you use schema startup, you must set up a custom URL scheme in info.plist.

  • Example

  <key>CFBundleURLTypes</key> 
  <array> 
    <dict> 
      <!-- identifier --> 
      <key>CFBundleURLName</key> 
      <string>routecode</string> 
      <!-- URL Scheme --> 
      <key>CFBundleURLSchemes</key> 
      <array> 
        <string>routecode</string> 
      </array> 
      </dict> 
  </array>

  • CFBundleURLSchemes

    • If you set the scheme parameter in the server’s resist token issuing API, it must match the value of the scheme parameter.

  • Host part

    • The value of the host part is unid_auth.

    • Please route it with the scheme in AppDelegate.

Automatic Key Registration

During this method’s execution, display a progress dialog to restrict user operations.

Automatic Login Authentication

Automatically performs Unid authentication.
On success, returns the userInfo parameter set in Automatic Key Registration.
Use userInfo to perform identity verification in login processing.
During this method’s execution, display a progress dialog to restrict user operations.
