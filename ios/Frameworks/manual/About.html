About iOS

Description

The method is the same for both Objective-C and Swift.

  • “./RouteCode.xcframework” is the main component.

How to Add the Framework

  1. Set it in the project

  2. App Target > General > Frameworks, Libraries, and Embedded Content Drag and drop the xcframework from the build location into this section in Finder.

Note: The framework can be placed anywhere, but since it is added with a relative path, if you move the program folder, you must also move the framework.

  1. Modify info.plist.

  1. Set up the schema launch as shown below.

  2. Ensure that the camera can be used as per the SDK specifications.

The text is as follows:

<?xml version="1.0" encoding="UTF-8"?><!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN"
    "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
    <dict>
        <key>CFBundleURLTypes</key>
        <array>
            <dict>
                <key>CFBundleTypeRole</key>
                <string>Editor</string>
                <key>CFBundleURLName</key>
                <string>routeauth2</string>
                <key>CFBundleURLSchemes</key>
                <array>
                    <string>routeauth2</string>
                </array>
            </dict>
        </array>
        <key>UIApplicationSceneManifest</key>
        <dict>
            <key>UIApplicationSupportsMultipleScenes</key>
            <false />
            <key>UISceneConfigurations</key>
            <dict>
                <key>UIWindowSceneSessionRoleApplication</key>
                <array>
                    <dict>
                        <key>UISceneConfigurationName</key>
                        <string>Default Configuration</string>
                        <key>UISceneDelegateClassName</key>
                        <string>SceneDelegate</string>
                        <key>UISceneStoryboardFile</key>
                        <string>Main</string>
                    </dict>
                </array>
            </dict>
        </dict>
        <key>NSCameraUsageDescription</key>
        <string>For SDK</string>
    </dict>
</plist>
